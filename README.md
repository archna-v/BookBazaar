# 📚 BookBazaar

A modern, responsive web application for browsing and managing a collection of books. Built with React and designed to provide an engaging user experience for book enthusiasts.

## ✨ Features

- 📖 Browse and search through a collection of books
- 🎨 Modern, responsive UI design
- ⚡ Fast and efficient performance
- 🔍 Advanced filtering and sorting options
- 💾 Local storage support for user preferences
- 📱 Mobile-friendly interface

## 🛠️ Tech Stack

- **Frontend Framework**: React (JSX)
- **Language**: JavaScript (99.5%)
- **Build Tool**: Vite
- **Styling**: CSS/SCSS
- **Package Manager**: npm/yarn

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v14 or higher)
- npm or yarn package manager
- Git

## 🚀 Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/archna-v/BookBazaar.git
   cd BookBazaar
   ```

2. **Install dependencies**
   ```bash
   cd frontend
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in browser**
   - Navigate to `http://localhost:5173` (or the port shown in terminal)

### Build for Production

```bash
npm run build
# or
yarn build
```

## 📖 Usage

1. **Browse Books**: Navigate through the book collection using the main interface
2. **Search**: Use the search functionality to find specific books
3. **Filter**: Apply filters to narrow down your book search
4. **View Details**: Click on any book to view more information

## 🎯 Key Components

- **Book List**: Displays all available books
- **Book Details**: Shows comprehensive information about a selected book
- **Search Bar**: Find books by title, author, or genre
- **Filter Panel**: Refine results based on various criteria
